MESSAGE À COLLER
----------------
Ce fichier est un faux positif. Il s'agit d'un widget de bureau open source
(code public : https://github.com/DIDEE-EMS/Widget_Claude) qui affiche la
consommation de l'API Claude de l'utilisateur.

L'exécutable est généré par PS2EXE, qui empaquète un script PowerShell dans un
exécutable .NET — cause classique de détections heuristiques génériques .NET.

Les comportements détectés par l'heuristique sont tous légitimes et documentés
dans le code source public :

  rien n'est envoyé ailleurs).

  afficher/masquer le widget.
- Add-Type : compilation à l'exécution de l'interface WPF.
- Blobs base64 embarqués : sprites GIF et sons WAV de l'interface.
- Fenêtre console masquée : c'est une application graphique WPF, pas une console.
- Exécutable non signé, sans réputation établie.

Le seul accès réseau est vers https://api.anthropic.com/api/oauth/usage
(endpoint officiel de consommation Anthropic). Aucune persistance malveillante,
